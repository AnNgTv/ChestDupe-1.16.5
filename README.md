# ChestDupe-1.16.5

A lightweight Minecraft mod for Fabric 1.16.5 designed to automate chest item duplication. It adds a "Dupe" button to the chest interface to quickly move items and potentially exploit server-side desyncs.

## 🚀 Features
- **Dupe Button**: Automatically moves all items from a chest to your inventory and closes the screen in one tick.
- **Dupe & Disconnect**: Moves all items and immediately disconnects, a common technique for many dupe glitches.
- **Simple UI**: Integrates directly into the standard chest (Generic Container) GUI.

## 🛠 Usage

1. Open any chest, large chest, or shulker box.
2. You will see two new buttons: **Dupe** and **Dupe & Disconnect**.
3. Clicking **Dupe** will attempt to shift-click all items into your inventory instantly.
4. Clicking **Dupe & Disconnect** will do the same but also kick you from the server immediately after sending the packets.

## 📦 Installation

1. Ensure you have **Fabric Loader** installed for Minecraft 1.16.5.
2. Download the latest release (or build from source).
3. Place the `.jar` file into your `.minecraft/mods` folder.
4. Launch the game.
